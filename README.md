# cjp2p-ruby
Ruby implementation of https://github.com/kermit4/cjp2p

message types it can handle:
Content send/receive
Peers send/receive
PleaseSendContent send/receive
PleaseSendPeers send/receive
PleaseReturnThisMessage receive
ReturnedMessage send
MaybeTheyHaveSome send/receive
